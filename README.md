# 美股 股息率排名（usdiv）

按市值分档的股息率排行榜，覆盖**美股**市场，输出自包含 HTML 页面。

## 功能

- 市值筛选（两档，美元基准）：
  - 美股：市值 > 1000 亿美元 ／ 500 亿 < 市值 ≤ 1000 亿美元
- 每个市值档、每个口径（TTM 最近 12 个月 ／ LFY 最近完整财年）取股息率最高的 **Top 30**。
- 表头对齐 A 股股息率排行任务：`排名 | 名称 | 代码 | 现价 | 总市值 | 每股分红 | 分红次数 | 股息率`（LFY 表额外含历史年股息率）。
- 市值胶囊选择器、TTM/LFY 双 Tab、可点击排序表头。

## 运行

依赖：腾讯自选股 `westock-data` / `westock-tool` 技能脚本（absolute 路径引用），以及 managed 的 node 与 python 运行时。

```bash
/Users/green/.workbuddy/binaries/python/versions/3.13.12/bin/python3 scripts/pipeline.py
```

脚本会重新抓取美股行情与分红历史、按市值分档并生成 `index.html`（站点入口，Vercel 静态部署默认读取根目录 `index.html`）。
日期动态化（TTM 起点 = 今日 - 365 天），可直接用于每日定时更新。

## 文件

- `scripts/pipeline.py` —— 抓取 + 构建一体化流水线（自包含）。
- `index.html` —— 生成的排行榜页面（自包含，站点入口，可直接打开 / 部署）。
- `scripts/data/` —— 运行时中间产物（行情/分红 JSON、代码清单），由流水线重新生成，不入库。

## 数据说明

- 美股总市值单位：亿美元。
- 股息率口径：TTM = 最近 12 个月每股分红 / 现价；LFY = 最近完整财年每股分红 / 现价。
- 对特殊分红导致的畸高历史股息率做了护栏（超出 TTM 2.5 倍或绝对值 >15% 时置空）。
- 部分股票（如通讯类、部分 MLP）接口无分红历史，对应「每股分红 / 分红次数 / LFY」显示「—」。
- 数据来源：腾讯自选股行情与历史分红接口，仅供研究参考，不构成投资建议。
